Front-end

Desenvolvo interfaces modernas e responsivas com HTML, CSS e Bootstrap, criando experiências de usuário intuitivas e visualmente atraentes. Utilizo um pouco de JavaScript para agregar interatividade, garantindo uma navegação fluida e de qualidade em todas as plataformas.

Back-end

Desenvolvo soluções de backend com Python, utilizando os frameworks Flask e Django. Realizo integrações de dados por meio de bibliotecas Python, com ênfase no MySQL como sistema de gerenciamento de banco de dados, utilizando o MySQL Workbench para modelagem e criação de bases de dados otimizadas.

Análise de Dados

Transformo dados em insights estratégicos usando Python, Pandas, SQL e técnicas de Data Science, com suporte de Jupyter Notebook e Google Colab. Estou ampliando meus conhecimentos com Power BI e outras ferramentas do ecossistema Python.

Tecnologias que utilizo

  • HTML5, CSS3, Bootstrap, JavaScript
  • Python (Django, Flask)
  • SQL (MySQL, MySQL Workbench)
  • Pandas, NumPy, Matplotlib
  • Jupyter Notebook, Google Colab, Power BI (em aprendizado)
  • VSCode
  • Git, GitHub
